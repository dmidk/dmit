dmit
=====
*dmit* is a python package which collects mainly meteorological tools to avoid replicating code.

How to install?
-------------------------------
*dmit*

.. code-block:: sh

   pip install dmit

How to use?
-------------------------------
Just install and import where needed.

.. code-block:: python

   import dmit
