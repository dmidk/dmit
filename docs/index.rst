=====
dmit
=====
*dmit* is a small package which collects mainly meteorological tools to avoid replicating code.

How to use?
-------------------------------
Just install and import where needed.

.. code-block:: python

   import dmit

Or use on command-line.

.. code-block:: sh

   python -m dmit.dmitio.convert -h
